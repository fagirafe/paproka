# Premium B2B SaaS Design Studio Website

Этот проект представляет собой современный, высокотехнологичный лендинг для B2B SaaS дизайн-студии, созданный с использованием передовых веб-технологий. Сайт включает в себя сложные анимации, адаптивную верстку и премиальный дизайн.

## Стек технологий

*   **Фреймворк:** [Next.js 14](https://nextjs.org/) (App Router)
*   **Стилизация:** [Tailwind CSS](https://tailwindcss.com/)
*   **Анимации:** [Framer Motion](https://www.framer.com/motion/)
*   **Иконки:** [Lucide React](https://lucide.dev/)
*   **Типизация:** [TypeScript](https://www.typescriptlang.org/)

## Особенности

- **Адаптивный дизайн**: Сайт корректно отображается на всех устройствах, от мобильных телефонов до широкоформатных мониторов.
- **Плавные анимации**: Использование Framer Motion для создания живого и премиального пользовательского интерфейса.
- **Micro-interactions**: Детально проработанные взаимодействия с элементами управления.
- **SEO Оптимизация**: Правильная структура HTML5 и мета-теги для лучшего индексирования.

## Как запустить проект локально

### 1. Предварительные условия

Убедитесь, что на вашем компьютере установлена [Node.js](https://nodejs.org/) (рекомендуется версия 18.x или выше).

### 2. Клонирование репозитория

Склонируйте проект к себе на локальную машину (замените `[REPO_URL]` на фактическую ссылку после создания репозитория):

```bash
git clone [REPO_URL]
cd paprika
```

### 3. Установка зависимостей

Установите необходимые пакеты с помощью npm:

```bash
npm install
```

### 4. Запуск в режиме разработки

Запустите сервер разработки:

```bash
npm run dev
```

После этого откройте [http://localhost:3000](http://localhost:3000) в браузере, чтобы увидеть результат.

## Сборка и деплой

Для создания оптимизированной сборки для продакшена выполните:

```bash
npm run build
```

Для запуска собранного приложения:

```bash
npm run start
```
